# Color collection

Eclectic collection of predefined color palettes, and web resources.

## Palette files

The directory `palettes` contain a collection of GIMP palette files (.gpl), and
can be read by programs like [Gimp](https://www.gimp.org/),
[Inkscape](https://inkscape.org/), [Gpick](https://www.gpick.org/).
